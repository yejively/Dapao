# Dapao

### 프로젝트 기간
2023.10.20 ~ 2023.11.20

### 개발 인원
6명(팀프로젝트)

### 역할
관리자페이지, 체험단 결제, 메인팝업

### 개발 의도
사용자가 쉽고 편리하게 중고거래를 할 수 있게 구현하고,

현존하는 p2p중고거래 사이트에 차별성을 이용한 수익 창출을 목적으로

체험단을 구현해 개발을 하였습니다.

### 개발 환경
SpringFramework 활용
